# 單字分類重組設計

**日期：** 2026-03-31  
**範疇：** 資料庫分類結構調整（純資料操作，不涉及功能開發）

---

## 背景

目前資料庫有 17 個分類、約 291 個單字，存在以下問題：

- 多個分類過於細碎（交通/出行、自然科學、財經/投資 各只有 3 個字）
- 食物相關拆成 4 個分類（食物、食物口味/口感、食物調味料、餐廳相關），有其中一個（食物）只剩 2 字且含有錯誤資料
- 身體相關拆成 2 個（身體部位、身體動作），可合併
- 銀行相關（49 字）與財經/投資（3 字）主題相近
- `演員` 被錯誤放在 `食物` 分類
- `未分類` 有 13 個零散單字尚未歸位

---

## 目標

將 17 個分類整併為 11 個主題明確的分類，並將所有單字歸入正確分類。

---

## 新分類架構

| 新分類 | 來源分類 | 預估單字數 |
|--------|----------|-----------|
| 動物 | 動物（原樣保留） | 33 |
| 國家 | 國家（原樣保留） | 19 |
| 衣物 | 衣物（原樣保留） | 24 |
| 情緒 | 情緒 + 未分類（體貼、優點） | 22 |
| 人體相關 | 身體部位 + 身體動作 + 未分類（剪指甲） | 29 |
| 飲食 | 食物 + 食物口味/口感 + 食物調味料 | 36 |
| 餐廳相關 | 餐廳相關（原樣保留） | 29 |
| 財經/金融 | 銀行相關 + 財經/投資 | 52 |
| 網路/電腦 | 網路/電腦（原樣保留） | 11 |
| 家居生活 | 家具 + 未分類（電風扇、蓮蓬頭） | 20 |
| 日常生活 | 交通/出行 + 自然科學 + 未分類其餘（含演員） | 15 |

**總計：11 個分類，291 個單字**

---

## 刪除的分類（11 個）

- 身體部位 → 人體相關
- 身體動作 → 人體相關
- 食物 → 飲食
- 食物口味/口感 → 飲食
- 食物調味料 → 飲食
- 銀行相關 → 財經/金融
- 財經/投資 → 財經/金融
- 家具 → 家居生活
- 交通/出行 → 日常生活
- 自然科學 → 日常生活
- 未分類 → 分配至各分類

## 執行方式

純 SQL 資料操作，透過 Neon MCP 工具執行：

1. 在 `categories` 資料表新增 5 個新分類（人體相關、飲食、財經/金融、家居生活、日常生活）
2. 將 `vocabulary` 資料表的 `category_id` 批次更新至對應新分類
3. 刪除空的舊分類

所有操作皆可透過 SQL transaction 執行，確保原子性。
